Second Brain

A personal database constructed using Obsidian and GitHub. Used store class notes and connect the concepts for review during exam season. The goal of this project was to centralize all of my notes on one platform to be accessed from any device. The GitHub repository automatically synchronizes across my devices every 5min to maintain backups and accessibility.

Home DNS

Raspberry Pi configured to be a home DNS server with Pi-Hole installed. The goal was to create a home domain to monitor network traffic and queries locally and remote. Includes a Pi-VPN installaion acting as a secure tunnel home to maintain network traffic and provide remote access to other machines on the private network from a distance.
